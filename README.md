# Spiders-Hunter
Gra oparta o winform. Polega ona na klikaniu na pająki w celu usunięcia ich z pokoju Homera Simpsona. Jeśli gracz chybi zbyt wiele razy, Homer wpadnie w panikę, a gracz przegrywa grę. Gracz ma mozliwość wyboru jednego z trzech dostępnych poziomów trudności.
